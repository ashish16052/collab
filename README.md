# Installation
.env file:
```
PORT="5001"
SECRET="123456"
DB="mongodb://localhost:27020/collab"
CLIENT_ID="google_oauth_client_id"
CLIENT_SECRET="google_oauth_client_secret"
CLIENT_URL="http://localhost:3000"
```

backend:
```
npm install
node index.js
```

Frontend:
```
npm install
npm start
open http://localhost:3000
```
